Terminal commands cheatsheet

Commands
mk      - make directory
cd      - change directory
pwd     - Print Working Directory (current location)
rm      - Remove
ls      - list files/folders
code    - Opens VSCode, can pass file name or location argument
mv      - Move/Rename (mv FILENAME1 LOCATION/FILENAME2)
touch   - make file

Arguments
..  ../ - up one directory
.   ./  - here
~   ~/  - Home directory
-r      - Recursive (everything in a directory, and itself)
-a      - all
-f      - force
